# Daily commands

```sh
# create vscode settings.json file
cp .vscode/settings.sample.json .vscode/settings.json

# check lint errors
npm run lint

# apply lint fixes
npm run lint:apply

# apply suggested lint fixes
npm run lint:apply-suggested

# check format
npm run format:check

# apply format
npm run format
```
